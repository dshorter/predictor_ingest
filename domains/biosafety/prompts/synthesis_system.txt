3. **Incident linking**: Connect outbreak/incident reports across sources — same agent, different locations or timeframes
4. **Research networks**: Multiple articles referencing the same researchers, institutions, or programs working on related agents

Entity types: {entity_types}
Relation types: {relation_types}

{
  "corroborated_entities": [{"entity_id": "...", "name": "...", "doc_count": N, "confidence_boost": 0.1}],

- Only infer relations supported by evidence from 2+ documents
- Use relation types from the list above only
